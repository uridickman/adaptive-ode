\section*{Homework 5: Adaptive Non-stiff ODE Solver}

\subsection*{Problem Formulation}

In this homework assignment, we aim to develop a solver for non-stiff ordinary differential equations. That is, ODEs of the form
\begin{equation}
	\vb{y}'=\vb{f}(t,\vb{y})
	\label{eq:general_ode}
\end{equation}
that satisfy $h\lambda\gg -\infty$.

I will use second-order Adams linear multistep methods, utilizing P(EC)E to reduce computational cost of implicit Adams-Moulton with Adams-Bashforth as the predictor method.
I will also use adaptive step size control to obtain a numerical solution to \eqref{eq:general_ode} within a target error tolerance.

\subsection*{Numerical Schemes}

A Linear Multistep Method is a numerical method which involves a linear combination of multiple steps to approximate the solution at the next time step.
For non-constant step size Adams methods, we simply integrate \eqref{eq:general_ode} across a single timestep of size $h_n$
\begin{equation*}
	\vb{y}_n=\vb{y}_{n-1}+\int_{t_{n-1}}^{t_n}\vb{f}(t,\vb{y}(t))\,dt
\end{equation*}

The Adams methods approximate this integral using a polynomial interpolation of $\vb{f}(t,\vb{y}(t))$ using the past values of $\vb{f}$.
One option for the interpolating polynomial is the Newton interpolation.
The Newton interpolating polynomial passing through a set of points at times $t_i$ with the coefficients computed by the divided difference table, defined recursively by
\begin{equation}
	\vb{f}[t_n,t_{n-k}]=\frac{\vb{f}[t_n,t_{n-k+1}]-\vb{f}[t_{n-1},t_{n-k}]}{t_{n}-t_{n-k}}
	\label{eq:DD_table}
\end{equation}
where $\vb{f}[t_n]=\vb{f}(t_n,\vb{y}(t_n))$.
The polynomial is computed with
\begin{equation}
	\phi(t)=\vb{f}[t_n]+\vb{f}[t_n,t_{n-1}](t-t_n)+\vb{f}[t_n,t_{n-2}](t-t_n)(t-t_{n-1})+\cdots+\vb{f}[t_n,t_{n-k}](t-t_n)\cdots(t-t_{n-k+1})
	\label{eq:newton_poly}
\end{equation}

For second order Adams-Bashforth, we interpolate the past two $\vb{f}$'s. Integrating, we have the Adams-Bashforth method for non-constant step size
\begin{align}
	\vb{y}_n&=\vb{y}_{n-1}+h_n\vb{f}_{n-1}&\quad\text{order 1} \label{eq:AB_1o} \\
	\vb{y}_n&=\vb{y}_{n-1}+h_n\vb{f}_{n-1}+\frac{h_n^2}{2h_{n-1}}(\vb{f}_{n-1}-\vb{f}_{n-2})&\quad\text{order 2} \label{eq:AB_2o}
\end{align}

For Adams-Moulton, we interpolate across the current $\vb{f}$ first, and then the past $\vb{f}$'s. Thus, the Adams-Moulton method is given by the implicit scheme
\begin{align}
	\vb{y}_n&=\vb{y}_{n-1}+h_n\vb{f}_{n}&\quad\text{order 1} \label{eq:AM_1o} \\
	\vb{y}_n&=\vb{y}_{n-1}+\frac{h_n}{2}(\vb{f}_{n}+\vb{f}_{n-1})&\quad\text{order 2} \label{eq:AM_2o}
\end{align}

In order to avoid the computational cost and complexity of implementation introduced by the implicit Adams-Moulton method, we instead use the Adams-Bashforth method to predict $\vb{y}_n$ that is required by $\vb{f}_n$ in the Adams-Moulton method.
The Predictor-Evaluate-Correct-Evaluate method first predicts $\vb{y}_n\approx \vb{y}_n^{(0)}$, which is only a good approximation in the non-stiff limit.
$\vb{f}(t,\vb{y})$ is then evaluated at $(t_n,\vb{y}_n^{0})$.
The Corrector method uses this approximation to determine $\vb{y}_n$, after which $\vb{f(t,\vb{y})}$ is evaluated at $(t_n,\vb{y}_n)$.
The second-order Predictor-Corrector algorithm is as follows:

\begin{align}
	&\text{Predict} \quad \vb{y}_n^{(0)}=\vb{y}_{n-1}+h_n\vb{f}_{n-1}+\frac{h_n^2}{2h_{n-1}}(\vb{f}_{n-1}-\vb{f}_{n-2})& \nonumber \\
	&\text{Evaluate}\quad \vb{f}_n^{(0)} = \vb{f}(t_n,\vb{y}_n^{(0)}) &\nonumber \\
	&\text{Correct}\quad \vb{y}_n=\vb{y}_{n-1}+\frac{h_n}{2}(\vb{f}_{n}^{(0)}+\vb{f}_{n-1}) & \label{eq:PECE_2o} \\
	&\text{Evaluate}\quad \vb{f}_n=\vb{f}(t_n,\vb{y}_n)& \nonumber
\end{align}

However, on the first time step, we only have the point $(t_0,\vb{y}(t_0))$, so we cannot use a two-step Adams-Bashforth method.
In this case, we determine the starting values by using the one-step Adams-Bashforth method as the predictor and the two-step Adams-Moulton method as the corrector.

\begin{align}
	&\text{Predict} \quad \vb{y}_n^{(0)}=\vb{y}_{n-1}+h_n\vb{f}_{n-1}& \nonumber\\
	&\text{Evaluate}\quad \vb{f}_n^{(0)} = \vb{f}(t_n,\vb{y}_n^{(0)}) & \nonumber\\
	&\text{Correct}\quad \vb{y}_n=\vb{y}_{n-1}+\frac{h_n}{2}(\vb{f}_{n}^{(0)}+\vb{f}_{n-1}) & \label{eq:PECE_1o}\\
	&\text{Evaluate}\quad \vb{f}_n=\vb{f}(t_n,\vb{y}_n)& 
	\nonumber
\end{align}

For constant step size, the previous equations are still used, but with the additional fact that $h_0=h_1=\dots=h_n$.
